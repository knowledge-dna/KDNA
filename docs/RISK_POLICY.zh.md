# KDNA 风险政策

## 风险等级

| 等级 | 风险 | 示例 | 要求 |
|------|------|------|------|
| R0 | 低 | 写作、笔记、个人效率 | Human Lock + 验证 |
| R1 | 中 | 管理、教育、商务沟通 | Feynman + Test Lab + 已知局限 |
| R2 | 高 | 关系、合规、财务决策 | 专家审核 + 强化警告 + 证据覆盖 |
| R3 | 受限 | 医疗、法律、安全关键 | 默认不得进入公共注册表 |

## 领域风险检测

Studio Core 会检测高风险关键词（医疗、法律、金融、安全相关），并自动触发风险等级审核。

## 风险重新分级

作者可提供证据申请重新分级，由注册表维护者审核。
