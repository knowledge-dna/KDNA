# 安全策略

## 报告漏洞

如发现 KDNA 工具体系安全漏洞，请私密报告至 security@aikdna.com，72 小时内响应。请勿公开提交 Issue。

## 范围

本策略覆盖 KDNA 工具体系（kdna-cli、kdna-core、kdna-studio、kdna-registry、kdna-website、kdna-vscode）。领域判断内容的安全由 KDNA 安全框架和风险政策管理。

## 支持的版本

| 工具 | 版本 | 支持 |
|------|------|:--:|
| @aikdna/kdna-cli | >=0.16.0 | 活跃 |
| @aikdna/kdna-core | >=0.3.0 | 活跃 |
| @aikdna/studio-core | >=1.0.0 | 活跃 |
