# Communication KDNA Example

A minimal example domain for communication coaching.

Run:

```bash
node ../../validators/kdna-lint.js .
```

## Four Questions

### 1. What does this domain judge?

Whether a message is clear, listener-focused, and actionable — or verbose, self-centered, and ambiguous. Helps AI agents distinguish effective communication from performative speech.

### 2. Where does it apply?

- Drafting emails, messages, or announcements
- Preparing presentations or talks
- Giving feedback or instructions
- Crisis or high-stakes communication

### 3. Where does it NOT apply?

- Creative writing (fiction, poetry)
- Personal journaling or diary entries
- Legal contracts or technical specifications where precision overrides clarity
- Conversations where relationship repair is the primary goal, not message clarity

### 4. How do I use it?

```bash
kdna install github:aikdna/kdna-communication
kdna validate .
```
